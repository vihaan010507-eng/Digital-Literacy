# Digital Literacy Awareness Quiz

##  Overview
This project is a simple quiz created using **Google Forms** to test basic awareness of digital literacy among students. It focuses on important topics like online safety, responsible internet usage, and knowledge of useful digital tools.

##  Objective
The main goal of this quiz is to:
- Promote awareness about digital literacy
- Help students understand safe and responsible online behavior
- Introduce useful platforms for learning and collaboration

##  Tool Used
- **Google Forms** – for creating and managing the quiz
- **link** - https://forms.gle/nDnowKnfc635SUxw9
- **response link** - https://docs.google.com/forms/d/1EOQVTyIITQbQ0GkWLBwqtTq027ryy-bSKl5a8ifJvN8/edit#responses

## Quiz Details
- Total Questions: 5  
- Question Types:
  - Multiple Choice  
  - Short Answer  

## Topics Covered
- Online safety  
- Password security  
- Responsible use of social media  
- Coding practice platforms  
- Cloud collaboration tools  

## Features
- Easy to access and user-friendly  
- Automatically collects responses  
- Can be used for surveys, quizzes, and feedback  
- Provides quick insights through responses
